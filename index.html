HTML5

HTML é uma linguagem de marcação de hipertexto que serve para exibir conteúdos de maneira estruturada na web. O HTML5 é uma versão da linguagem de marcação que traz como um dos seus maiores pontos, a semântica na estruturação do código, e outras possibilidades como elementos de vídeos, gráficos e áudios.

CSS3

CSS ou folha de estilo cascata (Cascading Style Sheets) é o que define a estilização do conteúdo estruturado no HTML. O CSS3 é a última versão de CSS, nele podemos contar com mecanismos avançado para criar animações em páginas HTML ou XML, e também para alterar o design destas.

Git

O Git é um sistema de controle de Versão muito utilizado pelos desenvolvedores para o controle de código fonte. Nesse sistema, é possível que várias pessoas mexam no mesmo código, mantendo as antigas versões armazenadas e ainda assim, o código atualizado.
